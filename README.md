# 货位编码工具

一个纯前端小工具，支持：

- 货位码编码配置（区域-货架-行号-列号）
- 导出货位码导入模板 Excel
- 导出货位码标签 PDF（单页单标签）

## 使用方式

1. 直接用浏览器打开 `index.html`。
2. 顶部维护业务字段（字段名、枚举值、符号）。
3. 中间维护区域和货架（支持新增、复制、删除）。
4. 设置每个货架的行数、列数和业务字段枚举值。
5. 底部点击导出按钮：
   - `导出货位码导入模板`：生成 Excel
   - `导出货位码标签 PDF`：打开弹窗配置后生成 PDF

## 说明

- 货位编码格式：`区域-货架-层级-货道`，例如 `A-03-05-06`。
- Excel 导出列包括：区域编号、货架编号、层级编号、货道编号、全部自定义业务字段。
- PDF 每页仅 1 个货位标签，适配“一页一张贴纸”的打印模式。
