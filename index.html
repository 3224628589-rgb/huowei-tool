使用教程

一、工具能做什么

在浏览器里配置区域—货架—层级—货道的货位结构，维护每个货位的业务属性；可导出 Excel 模板供外部填写或对接系统，也可从 Excel 回写到本工具；还能按当前配置批量生成货位码标签 PDF（含 CODE128 条码），适合打印贴纸。

二、货位编码怎么组成

每个货位对应一条编码字符串，由四段依次拼接而成（中间连接符可在 PDF 设置里改，默认短横线）：

  • 区域编号：区域名称，如 A
  • 货架编号：货架代码，如 01
  • 层级编号：行号，两位，如 01～99
  • 货道编号：列号，两位，如 01～99

示例（默认连接符）：A-01-05-06 表示 A 区、01 号货架、第 5 行、第 6 列货道。

PDF 里若把某连接符打成空格，标签上的可见文字会在该处换行；条码仍使用不含换行的连续字符串。

三、业务字段（货架维度）

在「货架维度业务字段定义」中可新增 / 编辑 / 删除字段。每个字段包括：

  • 字段名：导出 Excel 时的列名。
  • 枚举值：英文逗号分隔的选项列表。
  • 符号：与枚举一一对应，便于人工识别（如库区缩写）。

配置货架时，每个货架要为这些字段各选一个枚举值。

四、区域与货架

  • 新增区域：自动生成区域名（A、B、C…），可在卡片里改名。
  • 货架：可新增、复制、删除；每个区域至少保留一个货架。
  • 行数 / 列数：决定该货架下有多少个层级、货道，从而生成多少条货位。
  • 点击货架卡片可选中；配合右侧缩略图可快速定位。

五、区域缩略图

右侧面板按货架格子展示当前「视图字段」的取值；可收起 / 展开。

  • 单击货架格：选中并滚动到对应货架配置。
  • 拖拽框选：批量选中多个货架格。
  • Ctrl / Cmd + 单击：加减多选。
  • 有选中时右键：可批量把选中格的「当前视图字段」改成同一枚举值。

六、导出货位码导入模板（Excel）

  1. 点击「导出货位码导入模板」。
  2. 在弹窗中勾选要导出的列；用左侧 ⠿ 拖动手柄可调整列顺序（A、B、C… 列标签会随之更新）。
  3. 「全选 / 全不选」可快速开关；至少选一列才能导出。
  4. 生成的表第一行为表头，每一行对应一个货位；除系统四列外，其余列为自定义业务字段。

七、导入已编码货位 Excel

注意：导入会按 Excel 重建当前工具里的业务字段定义和区域 / 货架结构，相当于覆盖本地配置（导入前请确认已备份或无需保留当前编辑）。

  • 使用第一个工作表；第一行必须为表头。
  • 表头必须包含四列：区域编号、货架编号、层级编号、货道编号（列名需与导出模板一致）。
  • 其他列名会被识别为业务字段，并自动汇总枚举选项。
  • 层级、货道为数字；异常值会按规则修正并可能在「导入结果校验」里给出警告。
  • 空行、缺必填列的行会被跳过；导入结束后请查看弹窗中的摘要与明细。

八、导出货位码标签 PDF

  1. 点击「导出货位码标签 PDF」，在弹窗中设置标签尺寸、纸张尺寸、排布方向、条码与文字的左右/上下关系、编码字号、各段连接符等。
  2. 右侧为打印预览示意，与导出 PDF 排版一致。
  3. 确认后「生成 PDF」：每个货位单独一页，条码为 CODE128，便于一页一张标签纸打印。连接符设置仅影响标签文字显示，不改变条码内容。

九、数据保存在哪里

配置会保存在本机浏览器的 localStorage 中（含区域、货架、字段、导出列选项、PDF 设置、缩略图是否收起等）。换浏览器或清理站点数据后需重新配置；重要数据请通过 Excel 导出备份。
